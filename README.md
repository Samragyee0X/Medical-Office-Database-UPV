Medical Office Information System
A relational database project designed and implemented as part of the Database Design course at Universitat Politècnica de València (UPV), Spain — Erasmus+ Exchange Programme, 2024.

Completed by a team of 4 erasmus students, Team L2-3E-11

About
Given a written case study of a company's medical office, we designed and built a complete database system covering:

UML conceptual design → relational schema derivation → 3NF normalisation → Oracle SQL implementation
Manages departments, employees, daily consultations, annual medical revisions, test results, and materials
Extended with prescriptions, medicines, and patient allergy tracking

Tools

Oracle · Draw.io · StarUML · Oracle Data Modeler
